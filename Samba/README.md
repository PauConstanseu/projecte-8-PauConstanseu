# UD10 – Servidor de fitxers amb Samba 📁

## Introducció 🖥️🔁🪟

En el món de les xarxes, és molt habitual que convisquin equips amb diferents sistemes operatius. En aquesta pràctica treballarem amb **Zorin (Linux)** 🐧 i **Windows 11** 🪟, i l'objectiu és fer que puguin compartir fitxers entre ells de manera transparent.

Per aconseguir-ho, utilitzarem **Samba** 🔧, un programari lliure que implementa el protocol SMB/CIFS. Gràcies a Samba, un ordinador amb Linux pot actuar com a servidor de fitxers per a clients Windows, oferint recursos compartits amb diferents nivells d'accés.

Al llarg de la tasca, configurarem diversos tipus de recursos: una carpeta pública accessible per a tothom sense contrasenya 🔓, carpetes personals per a cada usuari 👤, i una carpeta compartida amb permisos diferenciats ✍️👀. També bloquejarem l'accés a fitxers amb extensió `.zip` 🚫📦 i aprendrem a accedir des de Linux a un recurs compartit per Windows.

Aquests coneixements són fonamentals en entorns de treball on barregen sistemes operatius diferents, ja sigui a empreses, institucions educatives o a casa mateix. 🏢🏫🏠
