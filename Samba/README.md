# UD10 – Servidor de fitxers amb Samba

## Introducció

Aquesta pràctica consisteix a configurar un servidor **Samba** a una màquina amb **Zorin (Linux)** per compartir fitxers amb un client **Windows 11**. Samba és una eina que permet la comunicació entre sistemes Linux i Windows mitjançant el protocol SMB/CIFS.

L'objectiu és aprendre a gestionar recursos compartits, controlar permisos d'accés, crear usuaris específics per a Samba i bloquejar determinats tipus de fitxer.
